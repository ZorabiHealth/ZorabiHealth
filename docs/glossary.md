# Glossary

An alphabetical list of the most important platform terms.

## A

### Appointment

A scheduled clinical visit between a patient and a doctor.

## D

### Dashboard

The main screen for a role. It shows the most relevant actions and data.

### Deepgram Token

A short-lived access token used to connect voice features.

## E

### Edge Function

A server-side function that runs near the database and handles background tasks.

## M

### Medication Reminder

A scheduled alert that helps a patient take medicine on time.

### Message Thread

A conversation between authorized users.

## P

### Pairing

The link between a patient record and another device or user that should receive related alerts.

### Prescription

A clinical medication order created by a doctor.

### Profile

The user’s stored identity and role data.

## R

### Realtime

Live updates delivered to the UI when data changes in the database.

## V

### Voice Log

A spoken note that is transcribed and stored.

## W

### Web Push

Browser-based notifications sent through a push subscription.
